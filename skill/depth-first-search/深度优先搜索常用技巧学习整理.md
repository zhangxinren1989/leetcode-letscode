## 技巧1：树的高效深搜防止重复搜索
```java
public void dfs(int cur, int father,List<Integer>[] tree) {
    for (int next : tree[cur]) {
        if (next != father) {
            dfs(next, cur, tree);
        }
    }
}
```
解释一下，cur是当前节点，father是父节点，tree是这颗树，开始传入根节点即可，father传-1吧

if (next != father)，就是这句，防止了重复搜索，使搜索只会向一个方向进行
